공통기능 로그인 일반 로그인 - 아이디/패스워드를 입력하여 로그인하는 기능 회원가입 후 아이디와 비밀번호를 입력하고 로그인 버튼을 눌러 로그인 한다.
로그인 성공 시 메인 대시보드로 이동된다.
로그인 실패 시 팝업으로 로그인 실패를 알려준다.		 　 LOG_01 　
[비밀번호]

- 표시버튼 클릭 시 입력 내용을 마스킹 없이 볼 수 있다, 숨김버튼 클릭 시 다시 마스킹된다.		 　
아이디 찾기 휴대폰 인증
이메일 인증		 회원가입 시 입력한 개인정보를 아이디를 찾는 기능 아이디 찾기는 사업자등록번호/휴대폰번호/이메일 을 통해 찾을 수 있다.
회원가입 시 입력한 이름, 사업자등록번호를 입력하고, 핸드폰 번호를 통한 인증이나 이메일을 통한 본인인증 방식을 선택하여 인증을 거치면 아이디를 찾을 수 있다.
비밀번호 찾기 페이지로 이동할 수 있다.
*이메일 인증은 인증 메일 발송 후 5분이내에 확인이 되지 않으면 인증이 되지 않으며, 하루에 최대 3회 메일인증이 가능하다.		 　 LOG_03 　
비밀번호 찾기 휴대폰 인증 회원가입 시 입력한 개인정보를 비밀번호를 찾는 기능 비밀번호 찾기는 사업자등록번호/휴대폰번호/이메일 을 통해 찾을 수 있다.
회원가입 시 입력한 아이디/이름을 입력하고 핸드폰 번호/이메일 중 본인인증 방식을 선택하여 인증을 거치면
새로운 비밀번호 설정으로 넘어간다.
새로운 비밀번호, 새로운 비밀번호 확인을 입력한 후, 변경한 비밀번호로 로그인하면 서비스 이용이 가능하다.
아이디 찾기 페이지로 이동할 수 있다.		 　 LOG_04 　
로그인 알림 팝업 - 로그인 실패 시 팝업으로 알려주는 기능 근로자가 로그인에 실패했을 시, 알림 팝업으로 알려준다. 　 LOG_05 　
자동로그인 - 로그아웃을 하지않고 재접속 시 로그인이 되는 기능 로그인 시 자동 로그인 CheckBox를 선택하면, 재접속 시 자동으로 로그인이 가능하다. 　 LOG_06 　
회원가입 회원유형선택 - 회원 유형을 선택하는 기능 회원가입 시 근로자/기업을 선택한다. 　 JOIN_01 　
　 기업회원 유형을 선택하는 기능 기업 회원가입 시 일반기업/용역중개사무소를 선택한다.
용역중개사무소의 경우 "사업자등록증 인력알선 종목 심사를 통해 인증받으실 수 있습니다." 안내
일반기업의 경우 사업자등록증 상의 업태에 따라 심사 후 입찰 아이콘을 보여줄 수 있다.		 　 　 　
약관동의 - 서비스의 이용을 위해 약관 동의를 요구하는 기능 사용자가 해당 서비스를 이용하기 위해서 동의해야할 약관들을 보여주고, 동의를 얻는다.
전체 동의 checkbox가 있으며, 아래 항목 중 하나라도 check 해제 시 전체 동의는 해제된다.
*이용약관(필수), 개인정보 수집 및 이용에 대한 안내(필수), 위치정보 이용약관 동의(선택), 마케팅 수신동의(선택)
* 마케팅 수신동의 동의 시, 이메일, SMS 중 선택하여 수신하게 된다.
위치정보 이용약관 해제 후 다음으로 넘어갈 시, 알림 팝업으로 알려준다.
* "위치정보 이용약관 해제 시, 워킹맵 서비스 이용에 어려움이 있습니다."
팝업에는 위치정보 이용약관 Checkbox가 있어 이전으로 돌아가지 않아도 check 할 수 있다.		 　 JOIN_02 　
기업 정보 입력 주소 API
국세정 API		 회원가입을 위해 기업 정보를 입력하는 기능 사업자등록번호, 기업명, 대표자명, 종목/업종, 주소를 입력한다.
사업자등록번호, 기업명, 대표자명, 종목/업종을 수기로 입력하며,
사업자등록번호의 경우 "000-00-00000" 형식인지 국세청 사업자등록정보 조회를 통해 확인한다.
위 3가지 항목은 회원가입 이후 기업인증 절차에서 워킹랩스 admin 계정에서 수기로 확인한다.
종목/업종은 전 직종으로한다.		 　 JOIN_03 　
관리자 정보 입력 휴대폰 인증 회원가입을 위해 관리자 정보를 입력하는 기능 아이디, 비밀번호, 비밀번호 확인, 담당자명, 휴대폰 번호, 이메일 입력 후 휴대폰 인증을 거친다.
아이디는 중복확인을 거쳐 사용유무를 판단한다.
*아이디 중복확인 시 탈퇴한 회원 일 경우 "사용할 수 없는 아이디입니다." 팝업창이 생성된다.

 * 한 사업자번호에 여러명의 관리자를 만들 수 있다.
* 관리자의 경우 동일인의 중복가입은 불가하다(사업자번호,이름,생년월일,전화번로 비교)		 　 JOIN_04

　 　 [항목]

HOME 맞춤 정보 박스 　 　 　 사용자의 상황에 맞게 4개의 탭이 나타난다.
공고 등록 탭 → 매칭중 → 매칭완료 → 근무현황 탭(근무예정, 근무 중, 근무종료)		 　 CUSTOM_01 　
대표매칭 설정 　 메인 대시보드에서 보일 대표 매칭을 설정하는 기능 메인 대시보드 화면에서 보일 대표 매칭을 설정할 수 있는 기능이다.
대표매칭은 기간 내에 있는 모든 매칭 중 선택할 수 있다.
대표매칭 설정 전 매칭 건수가 1개인 경우, 최초 매칭을 대표매칭으로 설정한다.
대표매칭 설정 전 매칭 건수가 여러개 일 시, 최근 매칭을 대표 매칭으로 설정한다.		 　 CUSTOM_02 　
매칭 등록 - 공고 등록 페이지로 이동하는 기능 서비스 내 매칭, 평가 서비스 등이 이루어지고있지 않은 경우(불러온 정보의 갯수가 없을 경우) 공고 등록 탭이 나타탄다.
클릭 시 공고 등록 페이지로 이동한다.		 　 CUSTOM_03 　
매칭 중 - 매칭 중인 공고를 보여주는 기능 대표매칭으로 설정된 공고 중 매칭 중 상태인 공고를 보여준다.
자세히보기 클릭 시 공고별 지원자관리-매칭별 지원자 리스트로 이동한다.
(마감일 < 현재날짜) or (마감일 >= 현재날짜 and 모집인원 <= 지원자수) 가 false		 　 CUSTOM_04 　
근무현황 - 공고별 근무현황을 보여주는 기능 근무시간을 반영한 상태를 보여준다.
* 근무 예정: 매칭 완료 ~ 근무시간 전
매칭완료는 모집기간이 마감되었거나 모집 인원 완료되었을 때 나타난다.
(마감일 < 현재날짜) or (마감일 >= 현재날짜 and 모집인원 <= 지원자수) 가 true
자세히보기 클릭 시 매칭 정보 자세히 보기 페이지를 보여준다.
모집인원 완료 시 자동으로 공고는 마감되며,
모집기간이 마감되었을 경우 매칭기간 연장 버튼이 나타난다.
* 근무 중: 근무시간 내
* 근무종료: 근무시간 후 ~ 근로자 리뷰 작성		 　 CUSTOM_05 　
리뷰 등록 - 근무 종료후 근로자를 별점을 통해 평가할 수 있는 기능 근무 종료 후 해당 근무지의 근로자를 확인하여 평가할 수 있는 리뷰 등록 탭이 나타난다.
근로 종료 시간이 지나면 push 알림이 간다.
별점과 키워드를 통해 평가할 수 있으며 해당 내용은 상호간 확인할 수 있음을 명시한다.
일정 별점이 넘을 시 근로자의 (매너 온도)에 영향을 준다.
별점은 근무 종료 후 15일 이내에만 평가가 가능하다.
추후 별점에 따른 기타 의견을 통해 별점별 리뷰 키워드를 다르게 할 수 있다.
근로자 평점(별점)은 필수이며, 추가의견은 선택으로 작성한다.
추가리뷰 작성란은 필수로 하며 최소 글자는 10자 이다.
추가리뷰 작성란은 비속어 삭제(필터링) 기능이 구현되어야 한다.		 　 CUSTOM_06

공지사항 리스트 중 하나를 누르면 상세하게 공지사항 게시글을 볼 수 있다.
공지사항은 워킹맵 ADMIN 계정에서 업로드 할 수 있다.		 　 CS_01 　
고객센터 카카오톡 상담 - 카카오톡 챗봇으로 연결되어 문의사항을 남길 수 있는 기능 카카오톡 챗봇을 통해 상담을 진행할 수 있다. 　 CS_04 　
약관 및 정책 서비스 이용약관 - 서비스 이용을 위한 약관을 안내하는 기능 이용약관, 개인정보 처리방침, 위치기반 서비스 이용약관, 마케팅 수신동의 약관을 확인할 수 있다. 　 CS_05 　
앱 설정 알림설정 - 앱의 push 알림을 설정하는 기능 push 알림을 기능별로 on/off 할 수 있다.
공고/광고 알림으로 항목이 나뉘며.
실시간 매칭 알림을 off 할 경우, 안내 문구가 표시된다.		 　 SET_01 　
버전정보 확인 - 앱의 버전을 확인하는 기능 사용자의 디바이스에 설치되어있는 앱의 버전정보를 확인할 수 있으며,
스토어에 등록되어있는 버전보다 설치된 앱의 버전이 낮을 경우 앱 접속 시 업데이트를 안내하는 팝업창이 나오며 스토어로 이동되어 업데이트를 진행한다.		 　 SET_02
